# Restaurant and menu
### Introduction
This is simple flask based web application which shows restaurants and their menu. This will allow to add, edit and delete restaurant & menuitem.

For the database, this app uses SQLAlchemy ORM.
### Features
* Written in Python

### What's included
```
restaurant-menu/
  |-- templates
        |-- deletemenuitem.html
        |-- editmenuitem.html
        |-- menu.html
        |-- newmenuitem.html
        |-- restaurant.html
  |-- dbsetup.py
  |-- projectrestaurant.py
  |-- restaurantmenu.db
```

### Requirement:
flask, sqlalchemy
